# Integr8 — Body Measurement Guide

Tutorial interativo de medidas corporais para clientes da Integr8.

## Estrutura do repositório

```
/
├── index.html              ← guia interativo (EN/PT, masculino/feminino)
├── images/
│   ├── peso_masculino.jpg
│   ├── peso_feminino.jpg
│   ├── estatura_masculino.jpg
│   ├── estatura_feminino.jpg
│   ├── pescoco_masculino.jpg   ← adicionar quando pronto
│   ├── pescoco_feminino.jpg
│   ├── cintura_masculino.jpg
│   ├── cintura_feminino.jpg
│   ├── quadril_masculino.jpg
│   └── quadril_feminino.jpg
└── README.md
```

## Como adicionar novas imagens

1. Salve a imagem com o nome exato listado acima dentro da pasta `images/`
2. Commit e push — a imagem aparece automaticamente no guia
